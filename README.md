# Tableaux - TP2 - Le maximum

## Consignes

Écrire une fonction `maximum` qui prend en paramètre un tableau de nombres et retourne le plus grand élément du tableau.

**Signature de la fonction :**

```ts
function maximum(tableau: number[]): number
```

## Exemples

- `maximum([10, 20, 30])` doit retourner `30`
- `maximum([5, 15, 25, 35])` doit retourner `35`
- `maximum([100])` doit retourner `100`
- `maximum([7, 3, 9, 2])` doit retourner `9`
- `maximum([])` doit retourner `null`
